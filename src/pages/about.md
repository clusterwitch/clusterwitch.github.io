# About This Site

This is a static site deployed on GitHub Pages using Eleventy (11ty).

## Technology Stack

- **Eleventy** — Static site generator
- **Nunjucks** — Template engine
- **Vanilla JavaScript** — No frameworks
- **PDF.js** — Client-side PDF parsing
- **Marked.js** — Markdown rendering
- **OpenAI / Anthropic API** — AI-powered resume tailoring

## Privacy

All file processing happens in your browser. No data is sent to any server except for the AI API call, which contains only the data you explicitly submit.
