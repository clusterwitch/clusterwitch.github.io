# Welcome to Resume Tailor

This is a portfolio site with an integrated AI-powered resume tailoring tool.

## Features

- **Portfolio Pages** — Showcase your work
- **Resume Tailor** — Upload your resume and job description; AI tailors it for you
- **100% Client-Side** — No backend, no server, no data stored
- **Privacy-First** — Your API key never leaves your browser
